## КРИТИЧЕСКИЕ ОСОБЕННОСТИ

### 🔴 MEGA #2 - БЛОКИ 11 И 12 ПОМЕНЯНЫ МЕСТАМИ
Физическая проводка блоков 11 и 12 перепутана. В коде они исправлены:
- **Команда BLOCK:11** → физические пины 34-37
- **Команда BLOCK:12** → физические пины 30-33

### 🔴 MEGA #2 - БЛОК 15 АКТУАТОР 3 ИНВЕРТИРОВАН
Третий актуатор блока 15 (средний, ближе к центру) имеет обратную проводку:
- **Act3 UP:** пин 51 (не 50!)
- **Act3 DOWN:** пин 50 (не 51!)

### 🔴 ПИНЫ 52-53 НЕ ИСПОЛЬЗУЮТСЯ НА MEGA #2
На Mega #2 пины 52-53 **не подключены**. Используются только пины 22-51.

## ЛОГИКА РЕЛЕ

```cpp
#define RELAY_ON   LOW   // Инверсная логика: LOW = включено
#define RELAY_OFF  HIGH  // HIGH = выключено
```

Все реле используют **инверсную логику**:
- Для включения актуатора → устанавливаем пин в LOW
- Для выключения актуатора → устанавливаем пин в HIGH

## ПРОТОКОЛ СВЯЗИ

**Формат команды:**
```
BLOCK:<номер>:<действие>:<длительность_мс>
```

**Примеры:**
```
BLOCK:15:UP:6000      - Блок 15 вверх на 6 секунд
BLOCK:11:DOWN:10000   - Блок 11 вниз на 10 секунд
BLOCK:12:STOP         - Остановить блок 12
ALL:STOP              - Остановить все блоки
PING                  - Проверка связи
```

**Ответы:**
```
ACK:15:UP             - Подтверждение
DONE:15               - Блок завершил работу
PONG                  - Ответ на PING
ERROR:message         - Ошибка
```
